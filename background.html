Hi! I’m Sasha — a smart contract developer who likes code that’s easy to read, reason about, and trust. My journey began with stubborn curiosity—late-night debugging, one green test, then another as tests began to tell the plot; invariants set the rules; and clever tricks were refactored into plain truth.

My rule is simple - it starts with security and ends with clarity. Threat models before demos; explicit failures over quiet bugs; tests that read like a story so future-me doesn’t swear at past-me.

Security isn’t a feature; it’s the compiler of my choices.
